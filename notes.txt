PROJECT: TODO APP
GUIDE TO SOLVE THE PROBLEM:
SETUP FOR THE SERVER SIDE:
1. Framework: Laravel 8, PHPv8
2. Database: MySQL
3. Design Pattern: Repository Pattern AND USE API RESOURCE RESPONSE
4. LARAGON FOR DEV ENVIRONMENT

SETUP FOR THE CLIENT SIDE:
1. ReactJS v18
2. Apply Hooks and Typescript

Features:
Able to manage todo lists like create/add, edit/update, view/display
Able to filter like Pending, ongoing and mark as completed/done
